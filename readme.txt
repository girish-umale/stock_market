Stock Market

The project is created in django. It has 3 apps authentication, listing and search.

Authentication takes care of the login, logout and sign up functionality.
Listing takes care of stock related information. We can add new stocks by using django admin panel. We can view all stocks.
Search enables us to find specific stock from the database.
Database used : sqlite3
